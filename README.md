# 🏃 Runner Game

A simple and fun endless runner game built with **Python** and **Pygame**. Control the player, avoid incoming obstacles, and try to beat your highest score!

---

## 🎮 Features

- 🏃 Smooth player running animation
- 🦘 Jump mechanics with sound effects
- 🐌 Animated snail obstacle
- 🪰 Animated flying obstacle
- 🌄 Background and ground graphics
- 🎵 Background music & jump sound
- 📈 Live score tracking
- 🏆 High score saved in a text file
- 🎬 Start screen and Game Over screen
- ⚡ Smooth gameplay at 60 FPS

## 🕹️ Controls

| Key | Action |
|------|--------|
| **SPACE** | Jump |
| **↑ Up Arrow** | Jump |
| **Mouse Click** | Jump |
| **SPACE (Game Over)** | Restart Game |

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Runner-Game.git
```

### 2. Navigate to the project folder

```bash
cd Runner-Game
```

### 3. Install Pygame

```bash
pip install pygame
```

### 4. Run the game

```bash
python main.py
```
